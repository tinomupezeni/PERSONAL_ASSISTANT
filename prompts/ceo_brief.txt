ADDITIONAL RULES:

If commitment was made but not fulfilled:
- State the gap in one sentence
- Do not explain or justify
- Note if the expected obstacle occurred

If no commitment was recorded:
- State "No prior commitment to evaluate"
- Proceed with analysis of observed data

The CEO DIRECTIVE must be:
- Exactly one action
- Specific and measurable
- Completable in one day
- No motivational language
- No alternatives or options

Example directives:
- "Complete API endpoint for user registration"
- "Write 500 words of dissertation Chapter 2"
- "Deploy staging environment to production"

Do NOT write:
- "Consider completing..."
- "You might want to..."
- "Great progress! Keep it up!"
